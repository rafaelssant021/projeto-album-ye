# Projeto Álbum YE

Acesse o projeto: https://rafaelssant021.github.io/projeto-album-ye/

## Sobre
Este projeto consiste em uma página web inspirada na estética de álbuns musicais, com foco em design visual e apresentação de conteúdo.
A proposta foi recriar uma experiência imersiva, simulando a apresentação de um álbum, explorando layout, tipografia e identidade visual.

---

## Conceito
O projeto foi inspirado na forma como álbuns são apresentados visualmente, trazendo uma interface minimalista e focada na experiência do usuário.
A ideia foi unir desenvolvimento web com elementos de design, criando uma página com identidade própria.

---

## Tecnologias utilizadas
- HTML5
- CSS3

---

## Funcionalidades
- Layout visual inspirado em álbuns musicais
- Estrutura organizada de conteúdo
- Uso de tipografia e cores para identidade visual
- Interface simples e imersiva

---


## O que eu pratiquei neste projeto
- Criação de layouts com foco em design
- Organização visual de conteúdo
- Uso de CSS para estilização avançada
- Construção de páginas com proposta estética
